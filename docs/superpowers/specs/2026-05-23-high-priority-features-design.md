## 1. Быстрый трекер настроения (Дашборд)

### Что добавляем
Новая карточка на дашборде между блоком приветствия и блоком статистики. Позволяет залогировать настроение в один клик — без перехода в дневник.

### UI
- Заголовок: «КАК ТЫ СЕЙЧАС?»
- 5 кнопок-эмодзи: 😔 Плохо / 😐 Так себе / 🙂 Норм / 😊 Хорошо / 😄 Отлично
- После нажатия: выбранный эмодзи подсвечивается, карточка показывает «Записано ✓» и сворачивается
- Если настроение уже записано сегодня — карточка показывает текущую оценку без кнопок

### Маппинг эмодзи → число
| Эмодзи | Значение |
|--------|----------|
| 😔 | 2 |
| 😐 | 4 |
| 🙂 | 6 |
| 😊 | 8 |
| 😄 | 10 |

### Компоненты
- `src/components/Dashboard/MoodCheckIn.tsx` — новый компонент
- Встраивается в `DashboardPage.tsx` между `dashboard-greeting` и `dashboard-stats`

### API
- `POST /api/mood` — тело: `{ score: number, note?: string }`
- Ответ: `{ id, score, created_at }`
- Если запись за сегодня уже есть — обновляет её (upsert по дате + user_id)

## 2. Тепловая карта настроения (Страница Прогресс)

### Что добавляем
Новая карточка на странице «Прогресс» над существующим линейным графиком. Показывает сетку дней текущего месяца, цвет ячейки = уровень настроения.

### UI
- Навигация по месяцам: `◀ Апрель | Май 2025 | Июнь ▶`
- Сетка 7 колонок (Пн–Вс), строки — недели месяца
- Цветовая шкала (4 уровня):
  - Нет записи: `#132c47` (фон карточки)
  - Плохо (1–4): `#1e4060`
  - Так себе (5–6): `#2a6080`
  - Хорошо (7–8): `#0891b2`
  - Отлично (9–10): `#38bdf8`
- При клике на день — показывает тултип с датой и оценкой
- Легенда снизу: Плохо ░▒▓█ Отлично

### Компоненты
- `src/components/Progress/MoodHeatmap.tsx` — новый компонент
- Встраивается в `ProgressPage.tsx` первым в `progress-grid`

### API
- `GET /api/mood/calendar?month=2025-05` — возвращает массив `{ date: "2025-05-01", score: 8 }[]`

## 3. Кризисная поддержка (Дашборд + Сайдбар)

### Что добавляем
**Карточка на дашборде** — внизу страницы, всегда видна. Плюс **ссылка в сайдбаре** внизу навигации.

### UI — карточка на дашборде
- Красный индикатор (dot) + заголовок «Кризисная поддержка»
- Текст: «Если тебе сейчас очень тяжело — ты не один. Обратись за помощью.»
- Две кнопки рядом:
  - 📞 **Телефон доверия** — `8-800-2000-122` (бесплатно, РФ), открывает `tel:88002000122`
  - 💬 **Поговорить с ИИ** — навигирует на `/chat` и создаёт новую сессию
- Стиль: тёмная карточка с красной обводкой `rgba(239,68,68,0.25)`

### UI — сайдбар
- Кнопка «🆘 Нужна помощь» внизу сайдбара, над кнопкой «Выйти»
- Цвет: `rgba(239,68,68,0.1)` фон, `#f87171` текст
- При клике — прокручивает дашборд до карточки кризисной поддержки (или переходит на `/`)

### Компоненты
- `src/components/Dashboard/CrisisSupport.tsx` — новый компонент
- Встраивается в `DashboardPage.tsx` последним элементом
- `Sidebar.tsx` — добавить кнопку перед `.sidebar-bottom`

### Без нового API
Компонент статический, не требует бэкенда.
